### Key Features <a name="key-features"></a>

Key Features of Awesome Books:

- Book Collection Management: Easily add and remove books with validation checks.
- Dynamic Navigation: Seamlessly navigate between book list, add book form, and contact sections.
- Local Storage: User-entered book data is automatically saved and persists across sessions.
- Real-time Date: Experience the thrill of real-time date updates powered by Luxon.
